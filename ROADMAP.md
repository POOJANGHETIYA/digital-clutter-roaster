# Roadmap

This roadmap is intentionally lightweight and will evolve with community input.

## Now (v1 hardening)

- Improve category confidence and explanation quality
- Improve duplicate detection strategy and transparency
- Tighten performance for large folder scans
- Improve empty/error states and guidance

## Next (v1.x)

- Add unit tests for analyzers and formatting helpers
- Add integration tests for scan -> results flow
- Improve export/report UX
- Add richer filtering and sorting in results

## Later (v2 exploration)

- Optional cleanup assistant mode (still user-confirmed)
- Better long-term organization recommendations
- Cross-platform analyzer packs (while preserving local-first behavior)

## Good first issues

Starter issues should be clearly scoped, documented and labeled:

- `good first issue`
- `help wanted`
